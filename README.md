中国医科院肿瘤医院挂号系统v1.0
=========================
#### 项目部署环境
系统环境有如下要求：
* PHP >= 5.4
* MCrypt PHP 扩展
* 需要为 app/storage 目录下的文件设置写权限。
* 从 PHP 5.5 版本开始，针对某些操作系统的安装包需要你自己手工安装 PHP 的 JSON 扩展模块。如果你使用的是 Ubuntu，可以通过,  apt-get install php5-json 命令直接安装。
